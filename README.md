### 🚀 Portafolio de Proyectos Destacados



#### 🚚 [Optimizador de Rutas Logísticas (VRP)](#)

Aplicación web espacial desarrollada para resolver problemas de asignación de flotas y minimizar tiempos/distancias reales de conducción.

* **Tecnologías:** Python, OR-Tools, NetworkX, OSMnx, Streamlit, Folium.

* **Impacto:** Integración de datos geoespaciales para la toma de decisiones logísticas eficientes.



#### 💼 [Simulador de Escenarios Salariales y Costos](#)

Herramienta financiera diseñada para proyectar el costo real de empresa y automatizar presupuestos de Recursos Humanos.

* **Tecnologías:** Python, Streamlit, XlsxWriter.

* **Impacto:** Permite iterar múltiples escenarios de contratación considerando topes imponibles (UF/UTM actualizados por API), leyes sociales y provisiones corporativas.



#### 📊 [Dashboard de People Analytics y Rotación](#)

Sistema de seguimiento histórico de dotación, enfocado en el análisis profundo de la fuerza laboral.

* **Tecnologías:** Python, Pandas, Altair.

* **Impacto:** Procesa bases de datos mensuales crudas (Data Wrangling) para calcular KPIs de rotación definitiva y generar visualizaciones de flujo de personal.



#### ⛏️ [Modelo Predictivo Operacional para Minería](#)

Algoritmo de Machine Learning diseñado para proyectar variables operativas y apoyar la estrategia del negocio.

* **Tecnologías:** Python, Scikit-Learn, Pandas.

* **Impacto:** Implementación de Random Forest con control estricto de hiperparámetros para mitigar *overfitting*, validado con métricas de error (MAE, MAPE).
